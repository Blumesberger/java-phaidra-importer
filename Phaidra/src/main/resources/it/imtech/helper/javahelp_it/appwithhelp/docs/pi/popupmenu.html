Menù di suddivisone in capitoli

  • Mediante tale menù è possibile creare la struttura del libro o dela collezzione

  • Tieni nota della corrispondenza tra il nome della sezione che vuoi creare e il nome del file
    (es. coperta / file001, frontespizio / file0 03, capitolo 1 / file005, ecc.)

  • Seleziona con il tasto sinistro del mouse una sezione (es. copertina, capitolo...) o una pagina, poi clicca il tasto destro e comparirà il menù sottostante:


  • Apri: Espande un capitolo visualizzando i file contenuti in esso.

  • Pagina Bianca: Aggiunge una pagina bianca sotto a quella selezionata.

  • Aggiungi file: Aggiunge una o più immagini da un percorso a scelta.

  • Anteprima in Phaidra: Mostra l'immagine in anteprima della pagina in Phaidra

  • Rinomina: Rinomina un capitolo o un immagine del libro o della collezione

  • Copia e Incolla: Copia o incolla un file da un capitolo all'altro

  • Taglia e Incolla: Taglia e incolla un file da un capitolo all'altro

  • Cancella: Elimina un file o un intero capitolo

  • Aggiungi: Aggiunge un capitolo al libro o una sottocollezione alla collezione.

  • Cancella: Elimina un file o un intero capitolo

  • Ripristina: Torna un passo indietro in caso di errore da parte dell'utente.

  • Metadati singoli (Solo per collezioni)
    • Crea-Vedi Metadati: Crea un nuovo tab con metadati specifici della pagina selezionata.

    • Unione Metadati: Elimina il tab di metadati singoli associando i metadati generici all'oggetto in questione.

    • Chiudi Metadati: Chiude il tab dei metadati singoli chiedendo di esportare i dati singoli per l'oggetto in questione.
