Perchè Phaidra

Nel sistema bibliotecario di Ateneo, in seguito alla realizzazione di diversi progetti di digitalizzazione è emersa l'esigenza di individuare un sistema unico per:

ITALIANO

  • Gestire il servizio di conservazione a lungo termine delle risorse digitali

  • Valorizzare la produzione intelettuale dell' ateneo, presente e passata, in un ambiente svincolato dai fornitori commerciali.

  • Gestire ampie collezzioni di oggetti multimediali con un sistema controllato di autorizzazioni.



La scelta è caduta sul progetto PHAIDRA dell' università di Vienna, con la quale è iniziata una proficua collaborazione a partire dal 2010.
