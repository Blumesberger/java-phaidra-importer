User Interface

  • When the application starts it will detect your server and available languages and will see the following window.



  • Through this operation, you can choose a destination (a server) and the language for the application to run.
    The application may not recognize the certificate server selected, in this case, an additional window will appear with the prompt for the acquisition of the certificate.

  • If confirmed the application will acquire the certificate and add it to the list of recognized certificates. After the procedure the application is restarted.
