Main Interface

Upper menu

  • File: With this menu you can access the current assistant (help), the information on the application and updating of this functionality or closing the application.

  • Language:Allows you to choose the language of the application, clicking on Italian, English or German language application will be updated in real time.

  • Import:Allows you to import the structure and / or metadata object saved previously.
    It also allows you to import a PDF file that will be divided into images and included in the structure of the object.

  • Export:It allows you to export the structure and / or metadata object.
    EPUB and PDF export functionality can be accessed only if the scan of an object book.

  • Upload:Upload Phaidra in a book or a collection.
    If object collection you can also load a single video.

Structure of the book or collection (left column)

  • Show page files / images / uploaded documents.

Sections and pages (Only visible for objects book):

  • The tab allows you to select the folder containing the book in PDF format to be loaded, plus the possible presence of OCR data.

Metadatas:

  • The form lets you enter the metadata of the book or collection subsequently loaded in Phaidra.
