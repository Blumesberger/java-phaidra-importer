Menu subdivision in chapters

  • Through this menu you can create the structure of the book or dela collezzione

  • Keep note of the correspondence between the name of the section that you want to create and the file name
    (es. coperta / file001, frontespizio / file0 03, capitolo 1 / file005, ecc.)

  • Select with the left mouse button a section (eg. Cover, chapter ...) or page, then click the right button and menu will appear below:


  • Open: Expand a section displaying the files contained in it.

  • Blank Page: Adds a blank page below the selected one.

  • Add file: Add one or more images to a location of your choice.

  • Preview in Phaidra: Show preview image of the page in Phaidra

  • Rename: Rename a chapter or a picture of the book or collection

  • Copy and Paste: Copy and paste a file from one chapter to another

  • Cut and Paste: Cut and paste a file from one chapter to another

  • Delete: Delete a file or an entire chapter

  • Add: Add a chapter to the book or a subcollection of the collection.

  • Restore: Go one step back in the event of error by the user.

  • Metadata individual (Only for collections)
    • Crea-Vedi Metadata: Create a new tab with specific metadata of the selected page.

    • Union Metadata: Delete the tab of each metadata associating metadata generic object in question.

    • Close Metadata: Close the tab metadata individual asking to export the data to the individual object.
