Workbench

  • Through this interface, you can define the path of the workbench, or the folder containing the images to be uploaded to the server.



  • By clicking on the "select" will appear interface selection workbook, seleziare folder to continue with the creation of the digital object.

The selected folder will be displayed in an interface based on the type of the selected folder

  • Case 1: The selected folder does not contain an object digitized previously, in this case will be prompted to create an object "Book" or "Collection"

  • Case 2: The selected folder contains a digitized object above, in this case you will be asked whether to import the previously saved structure and metadata.
