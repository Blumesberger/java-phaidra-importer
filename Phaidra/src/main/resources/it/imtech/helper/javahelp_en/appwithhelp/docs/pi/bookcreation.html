Following is a step by step procedure for creating a book

  • 1. All pages of the book must be placed in a folder and should be available in image format (recommended formats: TIFF and JPEG).

  • 2. Folder created is necessary to assign the page files a sequence order (the sort order is alphabetical followed by incremental numeric code).

  • 3. Starts Phaidra Importer, select Server, Language, and the workbook (previously prepared) that contains the pages of the book.

  • 4. Displays the pages of the book and you can change the sequence of pages, create or delete pages and chapters using the pull-down menu displayed with a right click of the mouse on one of the images visible in the structure of the object.

  • 5. Enter the metadata you want to export, and the book by selecting charge from the top menu.
