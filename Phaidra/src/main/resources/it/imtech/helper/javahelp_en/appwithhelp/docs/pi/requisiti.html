Minimum software

In order to install the application Phaidra Importer is necessary that the system which will be installed satisfies the following requirements:

  • Java Runtime Environment: 1.7.x
    The Phaidra Importer must be sent running through Java Runtime.
    Java can be downloaded from http://www.java.com/it/download/ (attention, the Java version must be consistent with the operating system of the PC)

  • Authorization to install programs on your computer

  • Access to Phaidra should be allowed is therefore necessary to have an account Phaidra


It is recommended to load books and collections only from computers connected to the University of Padua.
