Requisiti minimi software

Per poter procedere all'installazione dell'applicazione Phaidra Importer è necessario che il sistema nel quale verrà installato possegga i seguenti requisiti:

  • Java Runtime Environment: 1.7.x
    Il Phaidra Importer deve essere mandato in esecuzione tramite Java Runtime.
    Java si può scaricare da http://www.java.com/it/download/ (attenzione, la versione di Java deve essere coerente con il sistema operativo del pc)

  • Autorizzazione a installare programmi sul computer

  • L’accesso a Phaidra deve essere permesso è quindi necessario possedere un account Phaidra


Si raccomanda di caricare libri e collezioni solo da computer collegati alla rete dell’Università di Padova.
