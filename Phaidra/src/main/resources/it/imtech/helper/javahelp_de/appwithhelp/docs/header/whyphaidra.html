Warum Phaidra

Im Bibliothekssystem der Universität, nach der Umsetzung der verschiedenen Digitalisierungsprojekten es hat gezeigt, die Notwendigkeit, ein einzigartiges System zur Identifizierung:

  • Verwalten den Dienst der Langzeitarchivierung von digitalen Ressourcen

  • Steigern Sie die geistige Produktion der Universität, Vergangenheit und Gegenwart, in einer Umgebung frei von kommerziellen Anbietern.

  • Verwaltung großer Sammlungen von Medienobjekten mit einer gesteuerten Genehmigungssystem.



Die Wahl fiel auf das Projekt PHAIDRA Universität Wien, mit dem begann eine fruchtbare Zusammenarbeit aus dem Jahr 2010.
