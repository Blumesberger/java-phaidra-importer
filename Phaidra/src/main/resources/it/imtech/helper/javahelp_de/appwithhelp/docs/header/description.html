Universität Wien - Universität Padua

Steht für:. Permanent-Hosting, Archivierung und Indexierung von digitalen Ressourcen und Vermögenswerte
ist ein Managementsystem digitaler Objekte mit Funktionen der Langzeitlagerung.


  • Leicht: Die Beschreibung und Indexierung von digitalen Objekten sind einfach und schnell. Die Erregbarkeit jedes hochgeladene Asset wird durch eine permanente Verbindung garantiert.


  • Reiable: Die Anwendung basiert auf Software Fedora Commons Repository basiert. Das Format für die Metadaten übernommen und digitale Objekte garantieren die langfristige Archivierung und Interoperabilität mit anderen Systemen.


  • Safe: Dank den Definitionen der Bedingungen d 'Gebrauch und ein Lizenzsystem für Objekte geladen.
