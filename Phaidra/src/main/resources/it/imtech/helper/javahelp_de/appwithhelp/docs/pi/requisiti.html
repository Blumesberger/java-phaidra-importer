Mindest Software

Um die Anwendung zu installieren Phaidra Importeur ist notwendig, dass das System, das die folgenden Anforderungen erfüllt, installiert werden:

  • Java Runtime Environment: 1.7.x
    Der Importeur muss Phaidra geschickt durch Java Runtime laufen.
    Java kann heruntergeladen werden http://www.java.com/it/download/ (Achtung, die Java-Version muss mit dem Betriebssystem des PC sein)

  • Authorization, Programme auf Ihrem Computer zu installieren

  • Zugriff auf Phaidra erlaubt sein sollte ist daher notwendig, um ein Konto haben Phaidra


Es wird empfohlen, Bücher und Sammlungen nur von Computern an die Universität von Padua angeschlossene Last.
