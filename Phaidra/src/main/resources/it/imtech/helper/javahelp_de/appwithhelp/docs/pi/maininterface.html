Hauptschnittstelle

oberen Menü

  • Datei: Mit diesem Menü können Sie den aktuellen Assistent (Hilfe), die Informationen über die Anwendung und Aktualisierung dieser Funktionalität oder Schließen der Anwendung zugreifen können.

  • Sprache:Hier können Sie die Sprache der Anmeldung wählen Sie auf Italienisch, Englisch oder Deutsch Anwendung wird in Echtzeit aktualisiert werden.

  • Import:Hier können Sie die Struktur und / oder Metadatenobjekt zuvor gespeicherte importieren.
    Es erlaubt Ihnen auch, um eine PDF-Datei, die in der Struktur des Objekts in Bilder aufgeteilt und aufgenommen werden wird importieren.

  • Export:Es ermöglicht Ihnen, die Struktur und / oder Metadaten Objekt zu exportieren.
    EPUB- und PDF-Export-Funktion kann nur zugegriffen werden, wenn der Scan eines Objekts Buch.

  • Hochladen:Laden Sie Phaidra in einem Buch oder einer Sammlung.
    Wenn Objektsammlung können Sie auch ein einzelnes Video zu laden.

Aufbau des Buches oder Sammlung (linke Spalte)

  • Seite anzeigen files / images / hochgeladenen Dokumenten.

Profile und Seiten (nur für Objekte Buch sichtbar):

  • Die Register können Sie den Ordner, das Buch im PDF-Format enthält, die geladen werden, sowie das mögliche Vorhandensein von OCR-Daten auswählen.

Metadatas:

  • Das Formular können Sie die Metadaten des Buches oder Sammlung anschließend in Phaidra geladen eingeben.
