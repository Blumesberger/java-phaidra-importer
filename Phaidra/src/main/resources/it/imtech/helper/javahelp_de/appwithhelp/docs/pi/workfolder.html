Werkbank

  • Über diese Schnittstelle können Sie den Pfad der Werkbank zu definieren, oder der Ordner mit den Bildern, um auf den Server hochgeladen werden.



  • Mit einem Klick auf die "Select" wird angezeigt, Schnittstellenauswahl Arbeitsmappe, um wählen Sie Ordner mit der Schaffung des digitalen Objekts fortsetzen.

Ausgewählten Ordner werden in einer Schnittstelle angezeigt werden, basierend auf der Art des ausgewählten Ordners

  • Fall 1: Der ausgewählte Ordner kein Objekt zuvor digitalisiert, in diesem Fall enthalten, werden aufgefordert, ein Objekt zu erstellen "Buch" oder "Collection" werden

  • Fall 2: Der ausgewählte Ordner enthält eine digitalisierte Objekt oben, in diesem Fall werden Sie gefragt, ob die zuvor gespeicherte Struktur und Metadaten zu importieren.
